## API 路由功能说明

### chat/route.ts（对话接口）
接收用户消息，根据角色 ID 构建系统提示词，调用豆包大模型以 SSE 流式方式返回角色回复。同时维护基于 sessionId 的内存会话状态（对话历史、用户记忆、轮次计数），支持滑动窗口上下文（最近 15 轮），并从回复中解析 [MEMORY] 和 [IMAGE] 指令分别更新用户记忆和触发图片生成。

### tts/route.ts（语音合成接口）
接收文本和角色 ID，根据角色配置的音色 ID 调用 TTS 服务生成 MP3 语音（24kHz），返回音频 URL 供前端播放。

### image/route.ts（图片生成接口）
接收图像描述提示词，调用 AI 绘图服务生成 2K 无水印图片，返回图片 URL 供前端展示。
